Basic Syntax

Emphasis

  • *Italics* OR -Italics-
  • **Bold** OR --Bold--
  • Avoid creating Italics/Bold, add \ before - or *

Paragraphs

For a line break, add either a backslash \ or two blank spaces at the end of the line.

Headings

  • # heading 1
  • ## heading 2
  • ### heading 3

Blockquotes

  • > followed by an optional space.
  • including empty line need > followed by an optional space at beginning as well
  • can be nested

Lists

  • Unordered lists can use either *, +, or - as list markers
  • Ordered lists use numbers followed by . or )

Links

  • (url)
  • [text](url)
  • [text][id]
    ⋮
    [id]: url “title”

Images

  • ![alt](url)
  • ![alt][id]
    ⋮
    [id]: url “title”

Code

  • `content`
  • block: using 4 spaces, or place 3 ` on a line above and below the code block

Nested Lists

To nest one list within another, indent each item in the sublist by four spaces
